## Iniciando SERVER Nodejs
Abra a pasta API

```bash
npm install
renomear o .env.example para .env
node ace serve --watch
# ou
npm run dev
#ou
yarn dev
# ou
pnpm dev
```
Abra [http://localhost:3333](http://localhost:3333) em seu navegador.

## Iniciando ReactJS

```bash
npm install
npm run dev
# ou
yarn dev
# ou
pnpm dev
```

Abra [http://localhost:3000](http://localhost:3000) em seu navegador.
